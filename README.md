# Ahmet Ergurum — Academic Website

Academic website for Ahmet Ergurum, built from the same customized
[al-folio](https://github.com/alshedivat/al-folio) codebase used by
[Jerik Cruz](https://github.com/jerikdcruz/jerikdcruz.github.io).

The site includes:

- an academic biography;
- dissertation and research pages;
- publications generated from BibTeX;
- teaching and policy pages;
- a downloadable PDF CV;
- responsive navigation and light/dark themes.

## Preview locally

```bash
bundle install
bundle exec jekyll serve
```

Open <http://127.0.0.1:4000/ahmetpolsci/>.

## Publication

The included GitHub Pages workflow is manual-only. It will not publish the
site merely because code is pushed. Run the **Build and deploy Jekyll site**
workflow from the repository's Actions tab only after the preview is approved.
